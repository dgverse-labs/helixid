# HelixID

A decentralized identity monorepo built on Hedera and W3C DID/VC standards.

## Packages

- **helix-core** — Core cryptography, schemas, and utilities
- **helix-api** — REST API service for DID, VC, and VP operations
- **helix-sdk-js** — JavaScript/TypeScript client SDK
- **e2e** — End-to-end test suite
